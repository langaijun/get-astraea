## On-Page SEO Checklist

### Each God Page Should Have:
- [ ] Unique title (50-60 characters)
- [ ] Meta description (150-160 characters)
- [ ] H1 with god name
- [ ] Internal links to related gods
- [ ] External link to Wikipedia or mythology source
- [ ] Image with alt text
- [ ] Schema markup (Article + Person + FAQPage)
- [ ] CTA to take the quiz

### Blog Posts Should Have:
- [ ] Target keyword in title
- [ ] Meta description
- [ ] 1,500+ words
- [ ] Internal links to relevant god pages
- [ ] External authoritative sources
- [ ] Share buttons
- [ ] CTA to take the quiz
